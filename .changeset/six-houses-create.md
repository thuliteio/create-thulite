Update for create-vite 8.2.0

## BREAKING CHANGES

### What changed

This update aligns with create-vite 8.2.0, which includes significant internal architecture changes and dependency updates.

### Why the change was made

- To stay current with the latest create-vite upstream improvements
- To benefit from performance enhancements and bug fixes
- To ensure compatibility with modern Node.js versions and tooling

### How to update your code

If you're using `create-thulite` to scaffold new projects, no changes are required to your existing projects. This change only affects the project creation process itself.

For maintainers or those extending the templates:

- Review any custom modifications to template files for compatibility
- Test the scaffolding process with your custom templates
- Update any documentation referencing create-vite versions
